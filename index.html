Dashboard Arkos — Configuração

Cole a URL da Web App do Google Apps Script para conectar em tempo real com a planilha de pedidos.

A URL deve terminar em /exec. Abra no Chrome ou Edge.

Como publicar

1Planilha → Extensões → Apps Script → cole codigo_apps_script.js → salve
2Implantar → Nova implantação → App da Web → Acesso: Qualquer pessoa → Implantar
3Advanced → Go to (unsafe) → Allow → copie a URL /exec
